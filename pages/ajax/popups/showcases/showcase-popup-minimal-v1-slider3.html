Toronto Beer Festival Webapp

Semester 2 Group Project

For this project, my group and I were tasked with creating a web app concept for the upcoming Toronto Beer Festival. We researched and created personas to lay the foundation of our application. We based our web app's features on what potential users would be looking for when they are navigating a big festival such as this one. By the end of the project, we had created a user-friendly web app that was fully functioning.

  • School Project

    This project was the first web app that I have ever worked on.

  • Our Solution

    We wanted to address the main issues that current festivals such as the Toronto Beer festival face. Bad signage and lack of proper maps were the key issues that we encountered. We created an app that allowed users to search for the locations that they would like sto visit. It showed and described to them the location and contents of each tent in the festival.
