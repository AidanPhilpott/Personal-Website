International Application System Redesign

Group Project

The purpose of this project was to redesign Sheridan's International student portal. It had not been updated in years and it was lacking when it came to graphic and UX design.

My team and I began this project by gathering data. We analyzed data that was collected already by Sheridan College. We decided that Sheridan's data was not enough to tackle a project this big so we decided to create some of our own data. We looked at the systems from other schools websites. We also interviewed international students, Sheridan International Centre staff, and agents that help students apply to schools.

We decided to focus on making agent's lives easier. We did this by making it possible for agents to have multiple applications open at once, manage their open applications seamlessly, and save their progress with the applications that they have open. We solved the biggest problems that agents told us existed with the current system.
